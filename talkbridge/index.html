About

About TalkBridge

TalkBridge は、会話の流れを止めにくくすることを目指したアプリです。相手の話した内容を画面で確認しながら、返事は文字で見せたり、そのまま読み上げたりできます。説明的すぎないシンプルな構成で、初めてでもすぐ使い始められます。

Features

Features

Real-time transcript 相手の声をその場でテキストとして表示し、会話の内容を追いやすくします。
Typed reply with optional speech 入力した返事をそのまま見せることも、音声で読み上げることもできます。
Built for quick handoff 設定を深く開かなくても、その場で使い始めやすいレイアウトです。
